# Contributing to PassIntel

Thank you for your interest in improving PassIntel! As an open-source project, we welcome all kinds of contributions.

## How to Contribute
1. **Fork** the repository.
2. Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
3. Commit your changes with clear messages (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a **Pull Request** directed to the main branch.

## Areas for Improvement
- Adding more local password analysis algorithms.
- Developing a password generator feature in the UI.
- Improving the UI/UX animations.
